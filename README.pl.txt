Monitoruj stan systemu:
Temperatura GPU i CPU, obciążenie, skalowanie, użycie pamięci, użycie dysku, użytkownicy, przepustowość sieci Ethernet i Wi-Fi, GPIO i wiele więcej :)

Steruj z Homey:
Wyłącz lub uruchom ponownie system, zaktualizuj system operacyjny i kontroluj wyjścia GPIO.

Kontenery Docker:
Zatrzymywanie, uruchamianie i restartowanie kontenerów

Uruchamiaj polecenia CLI:
Z poziomu zaawansowanego flow możesz uruchomić dowolne polecenie (mam nadzieję, że wiesz, co robisz ;))
