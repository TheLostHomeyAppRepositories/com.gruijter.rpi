Overvåk systemstatusen:
GPU- og CPU-temperatur, belastning, skalering, minnebruk, diskbruk, brukere, ethernet- og wifi-båndbredde, GPIO, og mer :)

Kontroller fra Homey:
Slå av eller start systemet på nytt, oppdater operativsystemet, og kontroller utgangene til GPIO.

Docker-containere:
Stopp, start og restart containere.

Kjør CLI-kommandoer:
Fra et avansert flyt kan du kjøre hva som helst av kommandoer (jeg håper du vet hva du gjør ;))
